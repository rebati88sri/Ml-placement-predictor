# 🎓 Placement Predictor (Toy Project)

A simple supervised machine learning project that predicts whether a student will be placed or not based on **CGPA** and **IQ**.  
This is a toy project built for learning purposes, so the predictions are not highly accurate.

---

## 📌 Features
- Uses classification algorithms (Logistic Regression, Decision Tree, etc.)
- Input: CGPA and IQ
- Output: Placement prediction → *Placed* / *Not Placed*
- Beginner-friendly code structure
